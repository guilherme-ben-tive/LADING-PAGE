Projetos

Galeria Interativa em WebGL

Uma galeria 3D feita com HTML, CSS e JavaScript utilizando a biblioteca Three.js. O objetivo foi criar um espaço virtual para exibir imagens e objetos em 3D de forma interativa.

Aprendizado: Fundamentos de computação gráfica, manipulação de câmera, luz e objetos tridimensionais no navegador.

Participação: Projeto individual.

Editor de Imagens com Filtros CSS

Um mini-editor de imagens feito em HTML, CSS e JavaScript, onde o usuário pode aplicar filtros como saturação, brilho, contraste e desfoque em tempo real.

Aprendizado: Manipulação de estilos com JavaScript, uso criativo de filtros CSS.

Participação: Projeto em grupo com Maria Souza.
